☕ Coffee Shop Management System
A Windows desktop application to manage daily coffee shop operations — orders, billing, inventory, and product records — built with VB.NET and SQL Server.

🔧 Tech Stack

Frontend: VB.NET (Visual Basic .NET)
Backend / DB: Microsoft SQL Server
IDE: Visual Studio 2019
OS: Windows 7 / 8 / 10 / 11


✨ Features

Secure Admin Login
Add, View, Update & Delete Products (CRUD)
Order & Billing Management with PDF Bill Generation
Inventory & Stock Tracking
Expenditure & Inventory Report Generation
User-friendly Dashboard for Employees


💻 Requirements

Visual Studio 2019+
SQL Server (any edition)
RAM: 4GB+ | Processor: Intel i3 2.8GHz+
500MB+ free disk space


🚀 Setup

Clone the repo
Open the project in Visual Studio 2019
Import the provided .sql file into SQL Server
Update the connection string in the project to match your SQL Server credentials
Run the project — Login with Admin / Password


📦 Modules
Module Description
Admin Panel Manage products, users, and system settings
Billing Generate and print bills in PDF format
InventoryTrack stock levels and manage items
ReportsView expenditure and inventory reports
